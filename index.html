WebhookMind
WIP Go C++ SolidJS ScyllaDB

A self-hosted webhook delivery platform I'm building for teams who don't want to rely on third-party services. Ships with local AI file extraction so sensitive docs never leave your server.

  • Go ingestion layer with built-in retries, exponential backoff, and dead-letter queuing.
  • C++ extraction engine using Poppler + Tesseract + Whisper.cpp — runs fully local, no cloud API calls.
  • ScyllaDB handles high-throughput event logs, PostgreSQL handles relational data — each doing what it's actually good at.

Cloudflare Workers — Downtime Webhook Queue
Live Cloudflare Workers Node.js PHP

When our origin server went down, webhooks were just getting dropped. Built this Cloudflare Workers layer to buffer everything at the edge and replay it automatically once the server comes back up.

  • 800k+ messages queued across 40+ parallel workers — origin never gets slammed on recovery.
  • Automatic replay mechanism with configurable retry intervals and dead-letter logging.
  • Deployed globally on Cloudflare's edge — instant rollback if anything breaks.
